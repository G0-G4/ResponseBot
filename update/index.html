обновление программы

При выходе новой версии, будет выведена информация об обновлении после чего будет предложено ввести да или нет для выполнения обновления введите да/нет >>

После обновления старая версия программы останется в папке с постфиксом времени обновления. Также будет сохранен текстовой файл с информацией об обновлении.
