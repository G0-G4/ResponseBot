Файл настроек

  • database задает путь для файла с ответами. Файл сохраняется если передан параметр --sqlite или --csv
  • email ваша почта
  • key лицензионный ключ для работы программы
{
    "database": "settings/comments",
    "email": "example@mail.ru",
    "key" : ""
}
